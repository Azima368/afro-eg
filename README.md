# AFRO - متجر الملابس الفاخر 🛍️

موقع إلكتروني كامل لمتجر ملابس "AFRO" بتصميم احترافي وأنيق.

## 🎨 المميزات

### للعملاء:
- **الصفحة الرئيسية**: Hero section كبير + عرض الفئات + المنتجات المميزة
- **صفحة المنتجات**: فلاتر متقدمة (الفئات، المقاسات، الأسعار) + ترتيب
- **صفحة تفاصيل المنتج**: معرض صور + اختيار المقاس + إضافة للسلة
- **سلة التسوق**: إدارة المنتجات + كوبونات خصم + إتمام الطلب
- **نظام الطلبات**: نموذج كامل لمعلومات التوصيل والدفع

### للإدارة:
- **لوحة تحكم كاملة**: إحصائيات شاملة
- **إدارة الأقسام**: إضافة/تعديل/حذف الأقسام مع الصور
- **إدارة المنتجات**: إضافة منتجات مع:
  - صور متعددة
  - معلومات تفصيلية
  - مقاسات متنوعة (S, M, L, XL, XXL)
  - السعر والمخزون
  - تحديد القسم
- **عرض الطلبات**: مراجعة جميع الطلبات وتفاصيلها

## 🔐 الدخول للوحة الإدارة

### الطريقة السرية:
اضغط على شعار "AFRO" في أعلى الصفحة **5 مرات متتالية** خلال ثانيتين

### بيانات الدخول:
- **اسم المستخدم**: `admin`
- **كلمة المرور**: `admin123`

## 🎯 كيفية الاستخدام

### 1. فتح الموقع:
افتح ملف `index.html` في المتصفح

### 2. التصفح كعميل:
- تصفح الأقسام والمنتجات
- أضف منتجات للسلة
- أكمل عملية الشراء

### 3. الدخول كمدير:
1. اضغط على اللوجو 5 مرات سريعة
2. سجل الدخول بالبيانات أعلاه
3. أضف أقسام ومنتجات جديدة

## 🎨 الألوان المستخدمة

- **أسود**: `#1a1a1a` (الخلفية الرئيسية)
- **ذهبي**: `#D4AF37` (اللون الأساسي)
- **بيج نيود**: `#F5F5DC` (النصوص الثانوية)

## 📦 كوبونات الخصم

استخدم هذه الأكواد في صفحة السلة:
- `AFRO10` - خصم 10%
- `AFRO20` - خصم 20%
- `WELCOME` - خصم 15%

## 🚀 الصفحات المتوفرة

1. **index.html** - الصفحة الرئيسية
2. **products.html** - صفحة المنتجات مع الفلاتر
3. **product.html** - تفاصيل المنتج
4. **cart.html** - سلة التسوق وإتمام الطلب
5. **admin.html** - لوحة الإدارة (مخفية)

## 💾 التخزين

جميع البيانات محفوظة في `localStorage`:
- الأقسام
- المنتجات
- السلة
- الطلبات
- المفضلة

التصميم متجاوب بالكامل ويعمل على جميع الأجهزة.
